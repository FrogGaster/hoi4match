# Telegram-бот поиск тиммейтов Hearts of Iron 4

Бот для поиска тиммейтов в Hearts of Iron 4 с механикой как в DaVinci: показ одного пользователя за раз, лайк/дизлайк.

## Установка

1. Клонируй репозиторий или скопируй файлы
2. Установи зависимости: `pip install -r requirements.txt`
3. Получи токен у [@BotFather](https://t.me/BotFather)
4. Создай файл `.env` и добавь: `BOT_TOKEN=твой_токен`
5. Запуск: `python main.py`

## Команды

- `/start` — регистрация или вход
- `/search` — найти тиммейта (показывает карточки по одной)
- `/profile` — посмотреть свой профиль
- `/matches` — список взаимных лайков
- `/stats` — твоя статистика (просмотры, лайки, регистрация)

## Дополнительно

- **Жалобы** — кнопка 🚩 под карточкой
- **Лимит** — 50 просмотров в день (настраивается через DAILY_VIEW_LIMIT)
- **Поиск** — приоритет пользователям с твоего сервера
- **Уведомления** — еженедельный дайджест по воскресеньям
- **Админ-панель** — `/admin`, `/admin_broadcast`, `/admin_ban` (только для ADMIN_IDS в .env)
